Welcome to the Rock Paper Scissors Game API!

This is a portfolio project demonstrating a rewarded ad experience where players earn coins for winning games with a streak multiplier system.

Available API Endpoints:

GET /health - Health check
POST /api/users - Create new user
GET /api/users/:username - Get user information
GET /api/stats/:username - Get user statistics
POST /api/play - Play a game round
GET /api/leaderboard - Get leaderboard
GET /api/users/:username/games - Get user game history

Game Rules:

  • Rock beats Scissors
  • Scissors beats Paper
  • Paper beats Rock
  • Win streaks multiply coin rewards: 1x ‚Üí 2x ‚Üí 3x ‚Üí 4x (max)
  • Base coin reward: 10 coins per win
